# 用户指南

!!! note "注意"

    平台的所有功能必须登录后才能使用！

## 使用模型

平台预先在模型库中提供了多种模型供用户自主选择。

### 应用模型

上传数据文件后，点击 `应用模型` 按钮即可使用选中的模型对上传的数据文件进行预测。

## 训练模型

如果希望从原始数据训练模型，请选择……
